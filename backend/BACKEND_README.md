# Pixel Patches Backend API

## Setup and run

1. Create a `.env` file with the following variables:
   ```
   DB_URL=mongodb_url
   FRONTEND_URL=frontend_url
   PORT=port_number
   SESSION_SECRET=your_secret
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Run tests:
   ```bash
   npm test
   ```
